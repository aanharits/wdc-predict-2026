# F1 WDC Predictor 2026

This Machine Learning project uses Python to predict the Formula 1 World Drivers' Championship (WDC) winner for the 2026 season under the new regulations.

## Overview
This project is a Machine Learning experiment to predict the Formula 1 World Drivers' Champion (WDC) for the 2026 season. What makes this project particularly interesting is the challenge of the new 2026 technical regulations, where team performance typically shifts drastically.

The main goal is to see if historical data patterns from previous seasons (2024-2025) can be used to estimate race results in a completely new era of racing.
